# Chase Transfer Path — Research Working Draft

## Plain-English Summary

Chase's public securitization documents should be reviewed to determine whether receivables follow a path from JPMorgan Chase Bank, N.A. to a bankruptcy-remote transferor, then to the Chase Issuance Trust, and whether defaulted or removed receivables return first to the transferor before any reassignment to JPMorgan Chase Bank, N.A.

## Working Diagram

```text
JPMorgan Chase Bank, N.A.
(Account Owner / Originator / Servicer)
        │
        │ Receivables Purchase Agreement
        ▼
Chase Card Funding LLC
(Transferor / bankruptcy-remote entity)
        │
        │ Transfer and Servicing Agreement
        ▼
Chase Issuance Trust
(Issuing Entity)
        │
        │ Default / removal mechanics, if applicable
        ▼
Chase Card Funding LLC
(Transferor)
        │
        │ Reassignment mechanics, if applicable
        ▼
JPMorgan Chase Bank, N.A.
(Plaintiff only if reacquired or retained)
```

## Account-Level Proof Questions

1. Was the receivable ever transferred from JPMCB?
2. If not, what account-specific document proves retention or exclusion?
3. If yes, what documents show the transfer path?
4. If defaulted or removed, what document shows conveyance back to the transferor?
5. What executed reassignment returns the receivable to JPMCB?
6. Does the reassignment schedule identify the defendant's account?
