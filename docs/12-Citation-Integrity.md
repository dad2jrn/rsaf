# 12 — Citation Integrity

## 12.1 Core Rule

No citation may be used unless it is real, verified, and accurately characterized.

## 12.2 Required Citation Metadata

For each case, RSAF should know:

- Case name.
- Reporter citation or reliable database citation.
- Court.
- Year.
- Holding.
- Binding or persuasive status.
- Exact proposition supported.
- Limitations or adverse treatment, if known.

## 12.3 No Fabricated Quotations

Do not quote from a case, statute, regulation, or filing unless the quotation has been verified against the source.

## 12.4 Green v. Portfolio Recovery Associates

RSAF treats Green v. Portfolio Recovery Associates as a key Virginia authority on the insufficiency of account statements or generalized records, standing alone, to prove ownership where ownership is contested. Before citing Green in a filing, verify the exact citation, procedural posture, and holding.

## 12.5 Authority Classification

| Class | Description |
|---|---|
| A | Binding authority directly on point |
| B | Binding authority by analogy |
| C | Persuasive authority |
| D | Statutory or regulatory text requiring interpretation |
| E | Contractual/public document evidence |
| F | Analytical inference requiring express qualification |

RSAF should not present Class F reasoning as though it were Class A authority.
