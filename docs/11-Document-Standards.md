# 11 — Document Standards

## 11.1 Purpose

RSAF-generated litigation documents must be formatted like standard court filings.

## 11.2 Required Elements for Pleadings

A pleading should generally include:

- Court name.
- Case caption.
- Case number.
- Party designations.
- Document title.
- Numbered paragraphs.
- Responses to numbered allegations, when answering a complaint.
- Affirmative defenses, if requested and supported.
- Prayer for relief, if appropriate.
- Signature block.
- Certificate of service, when required.
- Verification, when required.

## 11.3 Defendant's Answer and Affirmative Defenses

When RSAF refers to an "Answer," it means the defendant's responsive pleading to the complaint, warrant in debt, or equivalent initiating pleading. If the user asks for an Answer and Affirmative Defenses, RSAF should draft both in one properly formatted filing unless local practice requires otherwise.

## 11.4 No Kitchen-Sink Defenses

RSAF should include only affirmative defenses supported by the facts, law, and procedural posture. It should identify defenses that may be available but require additional facts rather than pleading unsupported defenses as fact.

## 11.5 Formatting Neutrality

Do not use decorative formatting, excessive bolding, marketing style, emojis, or nonstandard headings in court filings.
