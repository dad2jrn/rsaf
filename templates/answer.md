# Template: Defendant's Answer and Affirmative Defenses

Receivable Standing Analysis Framework (RSAF)  
Version: 0.1.0  
Status: Skeleton

## Use

Use this template only when the user specifically requests a defendant's Answer or complete responsive pleading.

## Required Inputs

- Court name.
- Case number.
- Plaintiff name.
- Defendant name.
- Complaint or warrant allegations.
- Filing deadline.
- Jurisdiction.
- Whether affirmative defenses must be included with the Answer.
- Whether verification is required.
- Service method.

## Formatting Requirements

The completed Answer must include:

- court caption;
- case number;
- title;
- numbered responses matching the complaint;
- admissions, denials, or lack-of-knowledge responses;
- affirmative defenses if requested or required;
- signature block;
- certificate of service;
- verification if required.

## Drafting Rule

Do not include unsupported affirmative defenses. Do not use a kitchen-sink list. Identify defenses that may be waived if omitted.
